# YtYt - Школа программирования (Лендинг)

Ин як саҳифаи ягона (Landing Page) барои мактаби барномасозии **YtYt** мебошад. Лоиҳа дар асоси макети тарҳрезӣ сохта шудааст.

## 🛠️ Технологияҳо
Ҳангоми сохтани ин лоиҳа технологияҳои зерин истифода шудаанд:
* **HTML5** - барои сохтори сайт
* **SCSS / SASS** - барои стилҳо ва тартиби хуби код
* **Flexbox** - барои ҷойгиркунии унсурҳо ва мутобиқсозӣ (Responsive design)

## 🚀 Хусусиятҳо
* Стилҳо пурра бо препроцессори **SCSS** навишта шудаанд (бо истифода аз Nesting ва Variables).
* Мутобиқсозии содда (Барои экранҳои гуногун, аз ҷумла мобилӣ).
* Тарҳрезии тоза ва муосир мувофиқи макет.

## 💻 Чӣ тавр лоиҳаро дар компютер кушоем?
1. Репозиторияро ба компютери худ клонират кунед ё файли ZIP-ро боргирӣ кунед.
2. Файли `index.html`-ро дар кодери худ (VS Code) кушоед.
3. Тугмаи **Go Live** (Live Server)-ро пахш кунед, то саҳифа дар браузер кушода шавад.
